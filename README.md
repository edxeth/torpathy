# torpathy

A public `skills` package repo for the `torpathy` skill.

`torpathy` is a strategic programming decision skill for AI coding agents. It helps with architecture and fix-location questions by combining:

- **Karpathy** for explanation — incentives, defaults, feedback loops, protocol pressure
- **Torvalds** for prescription — invariants, contracts, APIs, ownership, lifecycle boundaries

## Install

```bash
npx skills add edxeth/torpathy
```

## Included skill

### torpathy

Strategic decision framework for programming and systems questions where the user is unsure how something should be done, is weighing valid approaches, wants to know where a fix belongs, or asks which side of a tradeoff dominates.

Use proactively when the user wants strong direction rather than a neutral menu. Works across backend, infra, frontend, APIs, databases, deployments, debugging strategy, agent workflows, and architecture tradeoffs.
