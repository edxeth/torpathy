# torpathy

A public `skills` package repo for the `torpathy` skill.

`torpathy` is a strategic programming decision skill for AI coding agents. It helps with architecture and fix-location questions by combining:

- **Karpathy** for explanation — incentives, defaults, feedback loops, protocol pressure
- **Torvalds** for prescription — invariants, contracts, APIs, ownership, lifecycle boundaries

## Install

```bash
npx skills add edxeth/torpathy
```

## Included skill

### torpathy

Strategic decision framework for broad programming and systems questions where the user is unsure how something should be done, is weighing valid approaches, or wants to know where a fix belongs.

Use it for:

- backend
- infra
- frontend
- APIs
- databases
- deployments
- debugging strategy
- agent workflows
- architecture tradeoffs
